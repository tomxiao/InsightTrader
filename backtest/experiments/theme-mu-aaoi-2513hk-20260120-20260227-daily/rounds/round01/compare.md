# round01

## 本轮目标

- 建立新主题的时间外 baseline，不以调优为首要目标。
- 在新的时间窗 `2026-01-20 ~ 2026-02-27` 中，验证当前 `round30` 形成的 `evidence + rules` 决策体系是否仍具有较高方向准确率。

## 跨标的总结

- 这轮完成了新主题的 full-round baseline 建立。统一口径下，全主题 `good / bad / unclear = 35 / 28 / 19`，方向准确率 **55.6%**。
- 这个结果很关键，因为它说明：
- 在更早的、完全不同的时间窗里，当前 `round30` 形成的 `evidence + rules` 决策体系依然维持了 `55%+`
- 也就是说，上一主题的突破并不只是单窗口偶然性，时间外稳定性已经初步成立
- 分标的结果为：
- `MU`: `10 / 7 / 11`，准确率 **58.8%**
- `AAOI`: `16 / 9 / 3`，准确率 **64.0%**
- `2513.HK`: `9 / 12 / 5`，准确率 **42.9%**
- 当前最主要的结构信号是：
- `MU / AAOI` 在新时间窗里仍然表现稳健
- `2513.HK` 则明显弱于原主题，是当前新主题的主要拖累

## 结构化归因观察

- `公司催化趋势型` 的准确率达到 **70.6%**，说明当前 evidence schema 与 rules 在美股趋势样本上的时间外稳定性非常好。
- `催化后高波动整理型` 为 **50.0%**，仍可接受，但比来源主题弱。
- `Beta情绪技术驱动型` 只有 **20.0%**，是这轮最差的分型桶。
- `看空` / `风险主导` 分支准确率为 **0.0%**，说明当前系统在这个时间窗里对 `2513.HK` 的防守型判断并不稳定。

## 当前结论

- 作为时间外 baseline，`round01` 是成功的：
- 它已经满足“新时间窗仍保持 `50%+`”这条泛化门槛
- 它甚至直接达到 **55.6%**
- 这意味着我们现在完全有资格进入你之前列出的第 3 个方向：锁定时间窗，更换标的做泛化验证
- 如果还想继续在本主题内部迭代，优先级也已经很清楚：只修 `2513.HK` 的 `Beta情绪技术驱动型` 和 `看空 / 风险主导` 残余误判
