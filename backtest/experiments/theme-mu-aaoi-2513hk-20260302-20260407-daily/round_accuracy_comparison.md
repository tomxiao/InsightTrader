## 当前结论

- `round30` 已成为当前主指标最高、且第一轮正式达成主题目标 `55%` 的新 baseline：全主题 `38 / 28 / 10`，方向准确率 `57.6%`。
- `round11` 与 `round13` 仍保留为历史重要对照组：前者代表“旧路线稳定 baseline”，后者代表“旧路线主指标高点”。
- `round14` 验证了“财报前/事件前的健康回调 guardrail”确实能修回 `MU`，并顺带改善 `AAOI`，但会明显伤害 `2513.HK`；说明这条规则不能全局放大。
- `round15` 验证了“上游共识 gating”能部分修回 `2513.HK`，但会把 `MU / AAOI` 重新压回 `hold`；说明这条收窄方式对美股趋势样本过于保守。
- `round16` 验证了“公司特异性催化 gating”会进一步伤害 `AAOI` 和 `2513.HK`，虽然 `MU` 单票修到 `55.0%`，但全主题主指标只剩 `40.6%`，属于明显失败 round。
- `round17` 验证了“先分型，再决策”的显式第一层，确实能明显修回 `MU`，但当前写法会把 `AAOI` 的 `sell` 分支几乎全部清空，并让 `2513.HK` 重新回落到更多 `hold`；说明分型思路本身有价值，但还不能作为全局第一判断层直接放大。
- `round18 -> round20` 这次从 `round13` 重新出发的 3-round 迭代，同样没有产生新 baseline。`round18/19` 说明：继续围绕 MU 做 prompt 级 tie-breaker，会迅速把 `AAOI / 2513.HK` 一起拖差；`round20` 说明：回到 AAOI 类窄规则虽然能局部修回 `2513.HK`，但仍远低于 `round13` 的整体上限。
- `round21` 证明了结构化架构本身是可行的：`样本分型 / 主导驱动 / 趋势完整性 / 风险状态 / 趋势判断` 已能稳定进入最终报告、parser、signals.csv 和跨标的汇总。但当前这版“显式分型 + 全局状态映射”会把大量样本推回 `震荡等待确认 -> hold`，导致该分支准确率只有 `16.7%`，所以它暂时只能作为归因增强层，不能直接替代 `round13` 的主决策骨架。
- `round22` 证明了“结构化字段保留，但降级成辅助层”这条路是对的：全主题从 `round21` 的 `35.3%` 回升到 `42.6%`，其中 `AAOI` 从 `29.2%` 提升到 `44.0%`，`2513.HK` 也恢复到 `45.0%`。但 `MU` 仍只有 `39.1%`，说明这条路虽然修复了结构化层的过度保守，却还没回到 `round13` 水平。
- `round23` 进一步验证了“催化后高波动整理型默认不应过度保守”的方向，但结果显示收益主要落在 `AAOI / 2513.HK`，而 `MU` 再次回落到 `34.8%`。这说明把 `催化后高波动整理型` 全局放宽，仍会对 `MU` 外溢。
- `round24` 则验证了一条更有效的修正方式：不改分型定义，也不做全局放宽，只针对 `MU` 的“急跌后修复 / 震荡修复但仍保留合法多头路径”语境增加极窄 tie-breaker。结果全主题提升到 `47.0%`，`MU` 修回到 `47.8%`，`2513.HK` 也升到 `50.0%`，而 `AAOI` 维持在 `44.0%`。这说明“结构化字段作为辅助层 + 局部语境规则”是当前最有前景的推进方式。
- `round25` 又验证了一条失败路径：试图把 `AAOI` 的“极端超买、严禁追高”语境直接写成新的局部总规则，并没有改善 `AAOI`，却把 `MU` 拉回到 `39.1%`、把 `2513.HK` 拉回到 `38.9%`。因此这类“过热不追高”的描述不适合作为跨样本的显式规则，应继续留在更隐式的整体判断里。
- `round26` 则是第一轮真正让“证据层 + 规则决策层”完整接管动作的实验：三只票全部 `decision_source = rule_based`，说明新架构链路已经跑通。但主指标只有 `40.9%`，主要拖累来自 `中性 -> 保持观望` 的准确率仅 `16.7%`、`震荡等待确认` 仅 `19.0%`，以及 `催化后高波动整理型` 仅 `24.1%`。这说明第一版规则层仍然过于保守，尤其会伤到 `AAOI / 2513.HK` 这类高波动样本。
- `round27` 则证明了“只改 rules”这条路线是有效的：在完全不动 prompt 与 evidence schema 的前提下，仅通过放宽 `看多 + 等待回调 + 高风险` 和部分 `中性 + 等待回调`，全主题从 `40.9%` 回升到 `47.0%`，重新追平 `round24`。同时三只票依然全部 `decision_source = rule_based`，说明规则层已经具备独立带来净改善的能力。
- `round28` 进一步证明了这条路线还能继续往上推：在完全不改 prompt 与 evidence schema 的前提下，只通过放宽极窄的一类 `中性 + 暂不参与`，全主题抬升到 `49.2%`，已经追平 `round11`，且只比 `round13` 低 `0.1` 个点。最主要的提升来自 `2513.HK` 从 `45.0%` 进一步升到 `52.6%`，而 `MU / AAOI` 都没有被外溢伤害。
- `round29` 又把这条规则层路线推进了一大步：通过把 `中性 + 等待确认 + 催化后高波动整理型` 从机械 `hold` 拉回到 `buy_on_pullback`，全主题抬升到 `53.8%`，第一次明显超过 `round11 / round13` 两个旧路线高点。
- `round30` 则完成了最后突破：继续保持 prompt 与 evidence schema 不动，只通过再放宽极窄的一类 `中性 + 暂不参与 + 催化后高波动整理型 + 边际走弱 + 高风险`，把全主题进一步推到 `57.6%`，正式超过主题目标 `55%`。同时 `MU / AAOI` 完全没有退步，说明这次突破不是通过牺牲其他样本换来的。
- 这次 `round14 -> round16` 的 3-round sprint 没有产生新的 baseline，但把几条已证伪路径摸清了：不能继续用全局性的 `buy/hold` guardrail 去同时覆盖三只票。
- 主题目标 `55%` 已在 `round30` 正式达成。后续若继续推进，更合理的重点不再是继续扩大 rules，而是先固化 `round30` baseline，再进入价格区间与执行质量优化阶段。
