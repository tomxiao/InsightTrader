# round08

## 本轮目标

- 基于 round07 的 `bad_case` 归因，重写三种状态的边界，而不是继续凭动作分布猜测该收哪一侧。
- 修复两类核心误判：
- `MU` 类：把“趋势中的健康回调 / 支撑区测试 / 偏多但等更好位置”误降级为 `震荡等待确认`。
- `2513.HK` 类：把“高风险但尚未形成明确下行主导”误升级为 `风险主导`。
- 保留 signal-first prompt 骨架，只改状态分界，不碰 analyst、parser 和 execution rules。

## 跨标的总结

- round08 在新的“未来 3 个交易日收盘均价”口径下，仍然是目前这条主题下最强的一轮纯信号 round。全主题纯信号标签为 `good/bad = 26/37`，方向准确率 `26 / (26 + 37) = 41.3%`；相对旧的第 3 日收盘口径，这轮 baseline 被明显抬高，也更稳健。
- 这次改善不是靠单一 ticker 硬拉，而是三只样本同时向好：
- `MU`：在均值版口径下达到 `9/11`，比原先收盘口径的 `7/15` 更能反映这轮对“趋势中的健康回调”识别能力的改善。
- `AAOI`：达到 `8/15`，依然是本轮增量最大的美股样本之一，说明“高风险但未必偏空”的修正不是偶然。
- `2513.HK`：来到 `9/11`，同时维持更均衡的动作分布和正收益执行结果，是港股 guardrail 仍然有效的最好证据。
- 从 prompt 归因看，这轮最有效的修改不是单纯“放松买入”或“收紧卖出”，而是把状态边界写清楚了：
- 趋势中的健康回调重新更容易落到 `趋势延续 -> 择机买入`
- 宏观风险、高波动、信息静默不再自动升级为 `风险主导 -> 建议卖出`
- round08 也暴露了下一阶段的新重点：方向质量已经明显改善，但执行层仍然拖累美股收益。`MU` 虽然纯信号回升到 `7/15`，但 `avg_return` 仍为 `-6.5776`；`AAOI` 的纯信号提升到 `8/17`，执行均值也仍只有 `-1.5098`。

- 结论：如果目标仍是“先优化信号，再优化价格区间”，round08 可以视为当前 signal-first 路线上的新 baseline。下一轮不应立即再改大的方向边界，而应优先选择：
- 要么做一次 `round09` 信号微调，只处理剩余少数 `MU / 2513.HK` 的边界误判；
- 要么正式进入第二阶段，把优化焦点切到价格区间与执行计划质量。
