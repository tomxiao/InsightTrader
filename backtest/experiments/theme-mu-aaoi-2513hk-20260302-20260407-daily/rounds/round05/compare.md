# round05

## 本轮目标

- 保留 round04 对 `MU / AAOI` 的“收弱买入、补风险主导卖出”主线。
- 只修一条 `decision manager` 边界：当高波动样本已经给出清晰支撑区、参与区和失效条件时，不要仅因“公司新闻静默 / 基本面不够漂亮”就机械压回 `保持观望`。
- 重点观察 `2513.HK` 是否能从过度保守中恢复，同时检查这条放松规则是否会把美股样本重新放宽。

## 跨标的总结

- 在新的“纯信号方向”口径下，round05 和 round04 的对比出现了更清楚的分化：`2513.HK` 的 `good/bad case` 并没有优于 round04，仍然是 `4 / 16`；这说明 round05 对港股样本的改善主要体现在执行结果，而不是信号方向本身。
- `MU`：`buy` 从 round04 的 `8` 升到 `9`，但纯信号标签下 `good/bad case` 从 `7 / 13` 变成 `6 / 13`。也就是说，round05 没有减少坏信号，只是少掉了一个好信号，方向上不如 round04 稳定。
- `AAOI`：`buy` 从 `11` 降到 `9`，`sell` 从 `2` 升到 `3`，但纯信号标签下 `good/bad case` 从 `5 / 20` 变成 `4 / 21`，是本轮最明确的退步样本。
- 结论：如果只看信号方向，round05 并没有优于 round04。`2513.HK` 至少没有继续恶化，但 `MU / AAOI` 都没有得到改善，其中 `AAOI` 还明显变差。
- 下一轮不应继续泛化这条例外，而应把它进一步收窄到更具体的场景：只有当价格已经真正回到关键参与区附近，且文本明确是“当前附近等待企稳参与”而不是“未来某次回调再说”，才允许从 `保持观望` 升到 `择机买入`。
