- 作为港股 guardrail，验证“政策 / 情绪 / 技术 beta 型”这条内部分型，能否在不要求公司特异性催化的前提下保住 round13 的反弹期参与能力。

## 改动内容

- 在 round13 骨架上增加“政策 / 情绪 / 技术 beta 型”内部分型，明确说明：不能因为公司新闻静默或基本面不够漂亮，就机械压回 `hold` 或 `sell`。

## 观察结论

- `2513.HK` 从 round13 的 `15/8/1` 回落到 `12/10/2`，准确率降到 `54.5%`。
- 说明这条 beta 分型虽然没有把港股样本彻底压坏，但也没能保住 round13 的最佳状态。
- 正向变化仍然存在：
- `2026-03-24`：`hold bad_case -> buy_on_pullback good_case`
- 但回撤更有代表性：
- `2026-03-23`：`buy_on_pullback good_case -> hold bad_case`
- `2026-03-30`：`buy_on_pullback good_case -> hold bad_case`
- `2026-03-03`：`sell good_case -> hold bad_case`
- 动作分布为 `buy=8 / hold=9 / sell=7`，说明它没有被压成极端保守，但“该参与的 beta 反弹被重新改成 `hold`”的问题又回来了。
- 执行层仍然维持正平均收益 `13.1859`，所以这轮对港股的伤害主要发生在纯信号层，而不是执行层完全失真。

## 下一步
