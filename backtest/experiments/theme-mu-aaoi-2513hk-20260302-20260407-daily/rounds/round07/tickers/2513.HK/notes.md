## 本轮目标

- 验证 signal-first prompt 能否修复此前港股样本因为执行计划约束而被过度压制的问题。

## 改动内容

- 移除旧 prompt 中大量与价格区间、执行可行性绑定的判断习惯，把核心问题改成“未来 `1-2` 周方向到底偏多、偏空还是不清楚”。
- 保留回测需要的 `截面价格 / 入场方式 / 失效条件` 字段，但明确它们不能反过来决定信号方向。

## 观察结论

- 这是 round07 最明确的改善样本。纯信号标签从 round04 的 `good/bad/unclear = 4/16/4` 提升到 `8/13/3`。
- 动作分布也发生了根本变化：`buy_on_pullback` 从 `3` 降到 `2`，`hold` 从 `15` 降到 `6`，`sell` 从 `6` 激增到 `16`。
- 这说明旧 prompt 里“执行计划不够完整就保持观望”的偏置被成功打掉了，但 signal-first prompt 又把大量原本的 `hold` 重新归到了 `sell`。
- 因此，这轮对 `2513.HK` 的结论不是“已经优化完成”，而是“方向判断确实改善了，但改善方式过于依赖偏空化”。

## 下一步

- 下一轮应保留 signal-first 骨架，不再回到旧的价格区间驱动逻辑。
- 重点修正的是 `sell` 的触发标准：不能仅因为上行动能不足或缺少确认就判成 `sell`，必须要求更清楚的结构破坏或风险主导证据，才能避免把 `2513.HK` 再次推成系统性偏空样本。
