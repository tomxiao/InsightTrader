# round29

## 本轮目标

- 保持 prompt 与 evidence schema 完全不变，只通过 `decision_rules.py` 继续修复 `round28` 剩余的 `中性 + 等待确认` 保守偏差。
- 本轮唯一实验假设是：
- `中性 + 等待确认 + 催化后高波动整理型` 中，有一批样本本质上已经更接近“高波动整理后的回调参与”，不应继续机械落到 `hold`。

## 跨标的总结

- 这轮是一次非常有效的 rule-only 改善。统一口径下，全主题 `good / bad / unclear = 35 / 30 / 11`，方向准确率 **53.8%**。
- 这意味着：
- 明显高于 `round28` 的 **49.2%**
- 首次明显超过 `round11` 的 **49.2%** 与 `round13` 的 **49.3%**
- 与主题目标 `55%` 只剩最后一小段距离
- 分标的结果为：
- `MU`: `11/10/5`，**52.4%**
- `AAOI`: `13/11/2`，**54.2%**
- `2513.HK`: `11/9/4`，**55.0%**
- 这轮第一次把三只票同时推到了 `50%+` 平台，说明新架构路线已经不只是“追平旧 baseline”，而是真正开始整体超越旧路线。

## 结构化归因观察

- 三只票继续全部 `decision_source = rule_based`，说明提升仍完全来自规则层。
- `等待确认` 的准确率从 `round28` 的 **16.7%** 大幅抬升到 **66.7%**，是本轮最关键的改善来源。
- `中性` 的准确率也从 **41.2%** 进一步提升到 **58.8%**。
- `趋势延续` 的准确率达到 **60.4%**，说明把这批高波动等待样本重新映射回 `buy_on_pullback`，并没有破坏整体买入分支。
- 当前剩余的主要拖累已重新收敛到两组：
- `看空 + 暂不参与 -> sell`
- 少量 `中性 + 暂不参与 -> hold`

## 当前结论

- `round29` 成为当前“证据层 + 规则决策层”路线的新最佳 candidate。
- 距离主题目标 `55%` 已经只差 `1.2` 个点，后续更适合继续做极窄的 rules-only 试验，而不是再回到 prompt 调优。
