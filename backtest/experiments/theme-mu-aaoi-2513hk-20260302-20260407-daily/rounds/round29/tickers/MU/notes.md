## 本轮目标

- 在继续只改规则层的前提下，验证 `中性 + 等待确认` 的修复不会对 `MU` 造成外溢伤害。

## 改动内容

- prompt 与 evidence schema 完全沿用 `round28`。
- 规则层只额外放宽了极窄的 `中性 + 等待确认 + 催化后高波动整理型` 组合。

## 观察结论

- `MU` 从 `10/11/5` 提升到 `11/10/5`，准确率从 **47.6%** 提升到 **52.4%**。
- 这说明本轮修复不仅没有伤到 `MU`，还顺手修回了一部分原本被压成 `hold` 的剩余 bad case。
- `MU` 当前主要拖累已经重新集中在一批 `看多 + 等待回调` 的错误买入，以及少量 `看空 + 暂不参与` 的误卖。

## 下一步

- 后续如果继续 rules-only 迭代，`MU` 仍应作为 guardrail 样本使用，不适合成为主攻对象。
