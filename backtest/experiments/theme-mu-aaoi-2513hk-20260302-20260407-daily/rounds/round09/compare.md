# round09

## 本轮目标

- 基于 round08 的均值版 `bad_case` 归因，继续做一轮纯信号微调。
- 重点想修 3 类问题：
- `AAOI` 的 `hold` 滥用，减少趋势性上涨中的持续观望。
- `MU` 的两段式误差：强势回升时不要压成 `hold`，转弱后不要继续给过密 `buy_on_pullback`。
- `2513.HK` 的反弹期误卖，继续压缩 `sell` 过多的问题，但不重新回到极端保守。

## 跨标的总结

- round09 没有超过 round08。按统一的未来 3 日均价口径，全主题来到 `good/bad = 25/39`，方向准确率 `39.1%`，低于 round08 的 `41.3%`。
- `MU` 是本轮唯一小幅改善样本：从 round08 的 `9/11` 提升到 `10/10`，同时 `avg_return` 也从 `-6.5776` 改善到 `-6.2515`，但提升幅度有限。
- `AAOI` 是本轮最明确的失败来源：从 round08 的 `8/15` 退回到 `5/17`。核心问题不是 `sell`，而是多天把原本已修回的 `buy_on_pullback` 再次压回 `hold`，重新引入了“趋势性上涨中持续观望”的坏模式。
- `2513.HK` 的纯信号略优于 round08，从 `9/11` 到 `10/12`，但代价很差：动作分布被重新推回 `buy=5 / hold=12 / sell=7`，实际成交重新掉到 `0`，说明这轮对港股样本又开始过度保守。
- 结论：round09 证明“把 round08 的剩余 bad case 一次性同时细修”并不稳。它在 `MU` 上略有收获，但重新伤到了 `AAOI` 和 `2513.HK`，因此不能取代 round08 作为 baseline。
- 下一轮不应继续沿 round09 这组 prompt 修正细磨。更合理的方向是回到 round08 baseline，只针对单一主问题开 round10，而不是同时兼顾三类误差。
