# round25

## 本轮目标

- 不动 `MU` 已经修回来的局部规则，只尝试修 `AAOI` 最大的一类坏样本：
- 当报告明确强调“极端超买 / 严禁追高 / 远离均线 / 天量成交后过热”，并把有效参与条件限定为深度回调后再评估时，尝试把该语境从 `趋势延续 -> 择机买入` 拉回更保守判断。

## 跨标的总结

- 这轮是失败 round。统一口径下，全主题 `good/bad/unclear = 27/39/10`，准确率 **40.9%**，明显低于 `round24` 的 **47.0%**。
- 分标的结果：
- `MU`: `9/14/3`，准确率 `39.1%`
- `AAOI`: `11/14/1`，准确率 `44.0%`
- `2513.HK`: `7/11/6`，准确率 `38.9%`
- 也就是说，这条新规则并没有修到 `AAOI`，却明显伤到了 `MU` 和 `2513.HK`。

## 当前结论

- “极端超买 / 不要追高”这类表述，虽然在 `AAOI` 的坏样本里常见，但它并不是一个足够独占的局部语境。
- 一旦把它显式升成 prompt 规则，就会被其他样本共享，重新扩大 `hold`，导致跨样本外溢。
- 因此 `round25` 不应继续沿用，工作区默认状态应回到 `round24`。
