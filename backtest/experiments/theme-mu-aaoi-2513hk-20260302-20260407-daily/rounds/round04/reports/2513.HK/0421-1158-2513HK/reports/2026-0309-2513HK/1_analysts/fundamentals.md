# 2513.HK 基本面分析报告

## 公司概况

北京智谱华章科技股份有限公司（简称“智谱”）是一家专注于认知智能大模型研发与服务的中国AI公司。

- **核心业务**：公司致力于打造新一代认知智能大模型，践行“模型即服务”（MaaS）的市场理念。
- **主要产品**：包括千亿级预训练模型GLM-130B、对话模型ChatGLM、开源模型ChatGLM-6B，以及智谱清言、CodeGeeX、CogVLM、CogView等AIGC产品矩阵。
- **商业模式**：通过大模型MaaS开放平台，为企业和开发者提供模型服务，推动AI技术商业化。
- **市场地位**：作为中国大模型领域的重要创新者之一，公司处于高增长、高投入的早期发展阶段。员工人数为1，094人，显示其作为技术研发公司的属性。

## 财务分析

### 资产负债表分析

截至2025年12月31日，公司总资产为48.54亿元，总负债高达129.62亿元，净资产为-81.08亿元，资不抵债，财务结构风险极高。

**关键资产项目**

1.  **短期存款：23.68亿元**
    这意味着公司持有大量现金及现金等价物，短期流动性储备看似充足。

2.  **应收帐款：6.99亿元**
    这意味着营业收入未能及时转化为现金，占用了营运资金，存在一定的回收风险。

3.  **物业厂房及设备：6.56亿元**
    这意味着公司在算力基础设施等固定资产上有重大投入，符合AI重资产投入的行业特性。

4.  **指定以公允价值记账之金融资产(流动)：3.77亿元**
    这意味着公司持有部分金融投资，可能用于现金管理或战略投资。

5.  **联营公司权益：3.38亿元**
    这意味着公司通过对外投资布局生态，但投资回报存在不确定性。

**关键负债项目**

1.  **衍生金融工具-负债(流动)：100.73亿元**
    这是一个异常巨大的数字，远超总资产。这意味着公司可能因发行可转换债券或含有衍生工具的融资工具，产生了巨额的公允价值变动负债，是导致净资产为负的核心原因。需要具体条款判断其是否为“纸面负债”。

2.  **应付帐款：13.29亿元**
    这意味着公司对供应商有大量欠款，利用商业信用支撑运营，但可能面临供应链压力。

3.  **短期贷款：6.05亿元**
    这意味着公司存在有息短期债务，带来明确的利息支出和偿还压力。

4.  **融资租赁负债(流动)：2.51亿元**
    这意味着公司通过融资租赁方式获取设备资产，构成长期付款承诺。

**权益与偿债结论**
- 股东权益为巨额负值（-81.08亿），主要受衍生金融工具负债影响，公司技术性破产。
- 净流动资产为-88.35亿元，流动负债远超流动资产，短期偿债压力极大。
- 财务结构极度脆弱，生存依赖于债权人的持续支持及衍生负债条款的最终结算方式。

### 现金流量表分析

2025财年，公司经营活动现金净流出22.46亿元，投资活动现金净流出3.76亿元，融资活动现金净流入25.99亿元，期末现金及等价物净减少0.23亿元。

**经营现金流：净流出22.46亿元**
这意味着公司主营业务“失血”严重，庞大的研发和运营开支远未通过自身收入覆盖，完全不具备“自我造血”能力。

**投资现金流：净流出3.76亿元**
这意味着公司仍在进行资本性开支（购建固定资产、无形资产）和对外投资（收购附属公司），处于扩张期。

**融资现金流：净流入25.99亿元**
这意味着公司高度依赖外部融资输血来维持运营和投资。现金流入主要来自新增借款（6.89亿）和发行股份所得（18.87亿）。

**现金净变动：净减少0.23亿元**
尽管融资流入巨大，但经营和投资流出规模更大，导致期末现金储备较期初微降。公司处于“融资驱动型”生存模式。

### 利润表分析

2025财年，公司实现营业额7.24亿元，但录得净亏损47.18亿元，每股亏损12.03元。

**收入表现**
- 营业额为7.24亿元。
- 缺乏同比数据，无法判断增长趋势。但从半年数据（2025H1：1.91亿）推断，下半年收入加速。

**盈利能力**
- 毛利为2.97亿元，毛利率约为41.0%。
- 这意味着公司核心模型服务具备一定的盈利空间，但被高昂的费用严重侵蚀。

**费用结构**
1.  **研发费用：31.80亿元**
    这是最大的开支项，是营业额的4.4倍。这意味着公司将所有资源倾注于技术研发，以保持竞争力，是战略性的巨额投入。

2.  **行政开支：5.05亿元**
    这意味着公司管理架构和后台支持成本高昂。

3.  **销售及分销费用：3.91亿元**
    这意味着公司正积极进行市场推广和渠道建设，以获取客户。

4.  **减值及拨备：0.22亿元**
    这意味着部分资产可能存在减值风险。

**亏损状况**
- 经营溢利亏损37.87亿元，除税前溢利亏损47.18亿元。
- 亏损主要来源于巨额研发投入，以及一项高达9.12亿元的“溢利其他项目”损失（可能与衍生金融工具的公允价值变动相关）。
- 公司处于典型的“战略性亏损”阶段，以巨额投入换取未来市场地位。

## 关键财务比率分析

基于现有年度数据计算关键比率（部分比率因净资产为负无意义）：

1.  **毛利率：41.0%**
    核心业务有毛利空间，但被费用完全吞噬。

2.  **研发费用率（研发/收入）：439.1%**
    极端高的投入强度，凸显其纯研发驱动属性。

3.  **经营现金流失率（经营现金流净额/收入）：-310.1%**
    每获得1元收入，经营现金净流出3.1元，现金流状况严峻。

4.  **资产负债率（总负债/总资产）：267.0%**
    资不抵债，财务杠杆风险极高。

5.  **流动比率（流动资产/流动负债）：0.29倍**
    流动资产远不足以覆盖流动负债，短期偿债能力极弱。

## 业务发展评估

### 积极因素

1.  **赛道前景广阔**：公司身处人工智能大模型核心赛道，市场空间巨大，符合长期科技发展趋势。
2.  **技术布局完整**：从底层大模型到上层应用产品矩阵布局清晰，具备全栈技术能力。
3.  **收入实现能力**：在巨额研发投入下，仍能产生7.24亿元营业额，证明其技术已具备初步商业化变现能力。
4.  **融资能力尚存**：2025年通过股权和债权融资获取近26亿元资金，显示资本市场仍愿为其故事提供“弹药”。

### 风险因素

1.  **生存风险**：净资产为负，经营现金流持续巨量流出，若无持续融资，公司将面临生存危机。
2.  **盈利遥遥无期**：研发等费用高企，收入规模短期内无法覆盖成本，扭亏路径不明，亏损可能长期化。
3.  **财务结构脆弱**：衍生金融工具负债规模巨大，其条款和最终结算对公司股权可能造成极大稀释或现金压力。
4.  **竞争白热化**：AI大模型领域巨头环伺，竞争异常激烈，持续的高研发投入是维持竞争力的必要前提，也是沉重负担。
5.  **流动性风险**：流动比率极低，短期偿债压力巨大，依赖再融资和债务滚动。

## 投资建议与风险提示

### 投资亮点
- **最强逻辑是“颠覆性技术的早期期权”**：投资智谱本质上是投资中国通用人工智能（AGI）的未来可能性。其技术积累和产品矩阵，使其成为该赛道有潜力的选手。

### 主要风险
- **财务破产风险**：当前资不抵债的状态是最大硬伤，任何融资中断或衍生工具不利结算都可能引发危机。
- **估值与现实的巨大鸿沟**：公司估值完全基于远期预期，与当前严峻的财务基本面脱节，股价波动会非常剧烈。

### 交易建议
- **适合的投资者类型**：极高风险偏好的成长型/风投型投资者；深刻理解AI技术、能承受本金全部损失、将其视为高风险科技期权的交易者。
- **不适合的投资者类型**：价值投资者、收入型投资者、风险厌恶者、寻求稳定股息或财务稳健标的的投资者。
- **持仓/建仓建议**：
    - **严禁重仓**，只可作为极高风险配置中的极小部分。
    - 建仓需紧密跟踪其融资进展、季度收入环比增长情况以及衍生负债的解决方案。
    - 未来1-3个月，任何新的重大融资消息或收入加速信号，可能成为股价催化剂。

## 结论

智谱华章是一家典型的高风险、高潜在回报的早期科技公司。其技术愿景和赛道站位具备吸引力，但当前的财务状况极其严峻，处于“技术性破产”和持续“失血”的状态。投资该标的完全基于对其技术成功和远期商业化的信念，而非任何当前财务指标的支撑。它是一张价格昂贵的“船票”，但船只本身正在漏水，投资者需清醒认知其中风险。

## 机会表达

- **当前基本面是否支持未来1-3个月的估值修复或预期改善**：**不支持**。当前基本面（资不抵债、巨亏、失血）无法提供支撑。估值修复完全依赖于新的、强有力的正面催化剂（如超大额融资、重磅产品突破、收入环比暴增）。
- **最强多头逻辑是什么**：公司是中国自主可控大模型技术栈的领先代表之一，若AI应用爆发或公司技术取得关键突破，其成长弹性巨大，是博弈AI产业浪潮的核心标的之一。
- **最关键证伪点是什么**：**融资断裂**。公司无法在未来6-12个月内获得新的股权或债权融资，导致现金流枯竭，生存危机爆发。
- **更像长期配置标的，还是阶段性交易机会**：**更像基于消息和情绪的阶段性交易机会**。在实现财务自给自足前，其股价将持续受融资进展、产品发布、行业政策等事件驱动，波动剧烈，不适合作为长期稳定配置。

**FINAL TRANSACTION PROPOSAL: HOLD (FOR EXISTING POSITIONS) / AVOID (FOR NEW INVESTORS)**
对于已持仓的投资者，鉴于其极高波动性和事件驱动特性，可在设置严格止损的前提下持有观望。对于新投资者，当前极端脆弱的财务基本面使得风险收益比不佳，建议回避，直至观察到融资环境改善或盈利路径清晰的信号。
