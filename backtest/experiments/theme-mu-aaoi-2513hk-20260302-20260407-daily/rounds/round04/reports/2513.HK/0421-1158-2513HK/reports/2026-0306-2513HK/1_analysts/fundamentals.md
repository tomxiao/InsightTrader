# 2513.HK 基本面分析报告

## 公司概况

北京智谱华章科技股份有限公司（智谱）是一家专注于认知智能大模型研发与应用的AI公司。

- **核心业务**：公司致力于打造新一代认知智能大模型，践行“模型即服务”（MaaS）理念。
- **主要产品**：包括千亿级预训练模型GLM-130B、对话模型ChatGLM、开源模型ChatGLM-6B，以及智谱清言、CodeGeeX、CogVLM等一系列AIGC产品。
- **商业模式**：通过大模型MaaS开放平台，为企业和开发者提供模型服务，推动AI技术在各行业的应用。
- **经营阶段**：根据财务数据，公司目前处于高研发投入、业务快速扩张的成长期，尚未实现盈利。

## 财务分析

### 资产负债表分析

截至2025年12月31日，公司总资产为48.54亿元，总负债高达129.64亿元，净资产为-81.10亿元，资不抵债状态显著。

**关键资产项目分析**

1.  **短期存款：23.68亿元**
    这意味着公司持有大量现金及现金等价物，流动性储备相对充足，能够支撑一段时间的运营和研发投入。

2.  **应收帐款：6.99亿元**
    应收账款规模较大，占流动资产比例较高，表明公司在业务拓展中给予了客户一定的信用账期，存在回款管理和坏账风险。

3.  **物业厂房及设备：6.56亿元**
    反映了公司在算力基础设施（如服务器等）上的重大资本投入，这是大模型公司的典型重资产特征。

4.  **联营公司权益：3.38亿元**
    表明公司通过股权投资布局生态链，但该部分资产的流动性和收益实现存在不确定性。

**关键负债项目分析**

1.  **衍生金融工具-负债（流动）：100.73亿元**
    这是资产负债表中最突出的项目，金额巨大。通常与公司发行的可转换债券或附有衍生工具特征的融资工具相关，其公允价值变动计入了负债。这意味着公司负债结构复杂，且价值波动可能对净资产产生巨大影响。

2.  **应付帐款：13.29亿元**
    高额的应付账款可能源于对供应商（如云服务商、硬件供应商）的采购，利用商业信用支撑了运营。

3.  **短期贷款：6.05亿元**
    表明公司使用了有息债务进行融资，将产生固定的利息支出压力。

**权益与偿债结论**
- 公司处于严重的资不抵债状态（净资产-81.1亿），主要原因是将巨额衍生金融工具负债计入当期。
- 尽管有24亿短期存款，但流动负债高达124亿，净流动负债达88亿，短期偿债压力极大。
- 财务结构风险极高，资产负债表非常脆弱。

### 现金流量表分析

2025财年，公司现金流状况严峻，主要依赖融资活动输血。

**经营现金流：净流出22.46亿元**
这意味着公司主营业务无法产生正向现金流，庞大的运营和研发开支远超过其业务收入带来的现金流入，处于“失血”状态。

**投资现金流：净流出3.76亿元**
流出主要用于购建固定资产、无形资产以及收购附属公司（合计约0.96亿），同时有大规模的投资与收回投资活动（净流出约3.13亿）。
这意味着公司仍在进行战略投资和资产构建，但投资活动整体是消耗现金的。

**融资现金流：净流入25.39亿元**
这是当年现金的唯一来源，主要包括新增借款6.89亿元和发行股份所得款项19.72亿元。
这意味着公司完全依赖外部股权和债权融资来维持运营和投资，属于典型的“融资驱动”型模式。

**现金净变动：净减少0.83亿元**
在巨额融资流入的情况下，期末现金及等价物仍较期初减少，凸显了经营和投资活动消耗现金的速度非常快。

### 利润表分析

2025财年，公司实现营业额7.24亿元，但亏损急剧扩大。

**收入表现**
- 营业额为7.24亿元。由于数据缺失去年同期对比，无法计算增速，但结合行业背景，收入应处于增长通道。

**盈利能力**
- 毛利为2.97亿元，毛利率约为41.0%。
  这意味着公司核心业务具有一定的盈利空间，但被高昂的费用严重侵蚀。

**费用结构**
1.  **研发费用：31.80亿元**
    这是最大的费用项，远超营业收入。这意味着公司正进行激进的研发投入以保持技术领先，是战略性的亏损。

2.  **销售及分销费用：3.91亿元**
    表明公司正在大力进行市场推广和渠道建设，以获取客户和市场份额。

3.  **行政开支：5.05亿元**
    反映公司日常管理和运营的支撑成本。

**亏损状况**
- 经营溢利为-37.87亿元。
- 股东应占溢利为-46.98亿元，每股基本盈利为-12.03元。
  亏损主要源于天量研发投入以及高达91.21亿元的“溢利其他项目”损失（很可能与衍生金融工具的公允价值变动相关）。
  这意味着公司处于“战略性亏损”阶段，净利润受非现金金融工具价值波动影响极大。

## 关键财务比率分析

1.  **毛利率：41.0%**
    显示产品/服务本身有溢价能力，但被期间费用完全覆盖。

2.  **研发费用率（研发费用/营收）：439%**
    极高，凸显公司技术驱动属性及当前阶段对研发的重度投入。

3.  **净利率：-648.8%**
    巨额亏损，盈利能力远未体现。

4.  **资产负债率：267%**
    严重大于100%，资不抵债，财务风险极高。

5.  **流动比率：0.29倍**
    流动资产远不足以覆盖流动负债，短期偿债能力极弱。

6.  **现金消耗率**：经营与投资活动合计净现金流出约26.22亿元。
    按当前融资所得现金消耗速度，现有现金储备支撑年限需密切关注。

## 业务发展评估

### 积极因素

1.  **赛道前景广阔**：身处AI大模型核心赛道，市场空间巨大，符合长期科技发展趋势。
2.  **技术布局完整**：拥有从底层大模型到上层应用的全栈产品矩阵，具备一定的生态竞争力。
3.  **融资能力尚存**：2025年成功通过股权融资近20亿元，显示资本市场对其技术路线和团队仍有信心，为持续研发提供了“弹药”。
4.  **营收实现突破**：已产生7.24亿元营业额，证明其技术具备商业化落地和变现潜力。

### 风险因素

1.  **财务结构极端脆弱**：资不抵债，流动比率极低，任何融资渠道收紧都可能引发严重的流动性危机。
2.  **盈利路径漫长**：巨额研发投入导致深度亏损，且亏损额远超营收，实现盈亏平衡遥遥无期。
3.  **估值与负债波动风险**：衍生金融工具负债规模巨大，其公允价值变动会剧烈影响利润和净资产，导致财报波动性极高。
4.  **行业竞争白热化**：面临国内外科技巨头的直接竞争，持续保持技术领先需要无底洞式的投入。
5.  **现金流高度依赖融资**：内生“造血”能力缺失，生存与发展完全取决于外部融资环境。

## 投资建议与风险提示

### 投资亮点
- **高弹性赛道核心标的**：作为纯正的大模型公司，在AI行业出现突破性进展或市场情绪转向乐观时，股价具备极大的向上弹性。
- **“关键生存资源”储备**：最新一轮股权融资提供了宝贵的现金缓冲，降低了未来1-2个季度的短期违约风险。

### 主要风险
- **资产负债表危机**：资不抵债是硬伤，可能引发监管关注、合作伙伴担忧，并限制其债务融资能力。
- **现金流断裂风险**：若下一轮融资不及预期，公司可能面临生存危机。

### 交易建议
- **适合的投资者类型**：风险承受能力极强的成长型/风投型投资者；擅长交易科技赛道趋势和波动的交易员。
- **不适合的投资者类型**：价值投资者、追求稳定股息收入的投资者、低风险偏好者。
- **持仓/建仓建议**：仅可作为高风险配置的极小仓位参与。更适合作为基于AI行业景气度的“交易性标的”，而非“配置型标的”。建仓需紧密跟踪其融资进展和月度经营数据。

## 结论
智谱华章是一家技术野心与财务风险并存的典型早期AI大模型公司。其技术布局具备长期想象空间，但激进的扩张策略导致其资产负债表严重恶化，陷入资不抵债且现金流高度依赖外部输血的困境。当前投资本质上是为其技术潜力和市场地位下注，同时承担其巨大的财务生存风险。它不适合绝大多数传统基本面投资者，而是专属于高风险偏好者的趋势博弈工具。

## 机会表达

**当前基本面是否支持未来1-3个月的估值修复或预期改善**
- **不支持**。未来1-3个月，基本面核心焦点是极高的财务风险，而非业务突破。资不抵债的状态难以在短期内扭转，且需要持续消耗现金，这更可能压制估值而非推动修复。

**最强多头逻辑是什么**
- 公司作为中国稀缺的纯正大模型标的，若AI行业出现重大技术突破、政策利好或市场风险偏好急剧提升，其股价将因极高的贝塔属性而出现大幅反弹。最新融资带来的“喘息窗口”是短期多头的主要依据。

**最关键证伪点是什么**
- **融资中断**。若市场环境恶化导致其下一轮股权或债权融资受阻，公司将直接面临现金流断裂和生存危机，所有长期逻辑将瞬间失效。

**更像长期配置标的，还是阶段性交易机会**
- **纯粹的阶段性交易机会**。其极端的财务结构和未经验证的盈利模式，使其不具备作为长期配置标的的稳定性基础。其股价波动将主要受行业情绪、融资消息和技术主题驱动。

**FINAL TRANSACTION PROPOSAL: HOLD (FOR EXISTING SPECULATORS) / AVOID (FOR NEW INVESTORS)**
对于已持仓的风险投资者，鉴于最新融资已提供短期安全垫，可暂时持有但需高度警惕融资面变化。对于新投资者，当前极高的基本面风险远超其可预见的回报，建议回避。
