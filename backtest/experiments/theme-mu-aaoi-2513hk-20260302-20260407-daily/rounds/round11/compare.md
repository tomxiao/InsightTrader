# round11

## 本轮目标

- 以 `round10` 为出发点，只修 `AAOI` 一类误差：
- 当公司级催化已经明显改变增长预期、正文又把当前回撤描述为消化涨幅/支撑测试/等待更好参与位时，不要过早从 `趋势延续 -> 择机买入` 降级成 `保持观望` 或 `建议卖出`。
- 同时要求不能明显破坏 `MU` 和 `2513.HK` 在 round10 已经得到的改善。

## 跨标的总结

- round11 是当前统一口径下表现最好的一轮。按未来 `3` 个交易日收盘均价的打标规则，全主题达到 `good/bad = 31/32`，方向准确率 `49.2%`，高于 round10 的 `45.5%` 和 round08 的 `41.3%`。
- `AAOI` 确实被修回来了：从 round10 的 `6/18/2` 回升到 `8/15/3`，重新回到 round08 的纯信号水平。最关键的修复是 `2026-03-19`、`2026-03-23`、`2026-04-02` 这三天重新回到 `buy_on_pullback good_case`。
- `MU` 没有因为专修 `AAOI` 被带坏，反而继续改善到 `11/10/5`。这轮把 `2026-03-04`、`2026-03-06` 的噪音买入收回成 `hold good_case`，同时把 `2026-03-13` 修成了 `buy_on_pullback good_case`。
- `2513.HK` 基本守住了 round10 的高位表现，从 `14/8/2` 变成 `12/7/5`。好样本略少，但坏样本也更少，准确率仍维持在 `63%+`，执行层反而提升到 `5` 笔交易、平均收益 `20.0955`。
- 结论：round11 不只是局部修补，而是第一次把 `AAOI` 拉回来的同时，还把全主题主指标进一步推高。它已经比 round10 更适合作为当前新的 candidate baseline。
- 距离主题目标 `55%` 仍有差距，下一轮更合理的方向不再是继续大改全局，而是围绕 round11 剩余坏样本做更窄的定向修正。
