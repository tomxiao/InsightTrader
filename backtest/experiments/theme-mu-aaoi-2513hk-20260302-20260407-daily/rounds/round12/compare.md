# round12

## 本轮目标

- 以 `round11` 为出发点，只验证一条更窄的 `AAOI` 支撑测试规则：
- 对“催化后高波动整理”样本，只有当当前价格已经真正进入报告认可的支撑测试区，才保留 `趋势延续 -> 择机买入`；若认可的参与区仍明显低于当前价格，则不要只靠旧催化维持偏多。
- 同时要求不能明显破坏 `MU` 与 `2513.HK` 的 round11 改善。

## 跨标的总结

- round12 是一次明确失败的 round。按统一的未来 `3` 个交易日收盘均价口径，全主题从 round11 的 `31/32` 回落到 `24/40`，方向准确率从 `49.2%` 掉回 `37.5%`。
- `AAOI` 没有被修好，反而从 round11 的 `8/15/3` 退回 `6/17/3`。本轮新增规则把多个本来正确的支撑附近 `buy_on_pullback` 直接压回 `hold`，例如 `2026-03-19`、`2026-03-20`。
- `MU` 也被一起拖回去了：从 round11 的 `11/10/5` 回落到 `10/10/6`。它被明显推向更保守的动作分布，`buy` 从 `13` 降到 `7`，`hold` 从 `7` 升到 `12`。
- `2513.HK` 受伤最重，从 round11 的 `12/7/5` 大幅退回 `8/13/3`。多处原本正确的反弹期 `buy_on_pullback` 被压成 `hold bad_case`，说明这条“当前是否已进入支撑测试区”的规则在港股高波动样本上外溢得过于保守。
- 执行层也同步恶化：`AAOI` 平均收益从 `-1.6337` 走弱到 `-4.8631`，`2513.HK` 从 `20.0955` 掉到 `4.8077`，`MU` 虽略优于 round11，但仍显著为负。
- 结论：round12 证明了“支撑测试 gating”这条路虽然直觉上像是在修 `AAOI`，但实际会把 signal-first prompt 重新推回“谨慎观望优先”，并且外溢伤害 `MU / 2513.HK`。这条改法不值得继续细磨。
