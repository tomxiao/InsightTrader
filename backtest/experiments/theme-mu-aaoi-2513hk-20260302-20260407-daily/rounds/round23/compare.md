# round23

## 本轮目标

- 在 `round22` 的“结构化字段辅助层”框架上，只修一个剩余坏模式：
- 不再让 `催化后高波动整理型` 因为“高波动 / 首轮回撤 / 不宜追高”就默认落入 `hold/sell`
- 验证这条放宽是否能继续抬升 `AAOI / 2513.HK`，同时不明显伤到 `MU`

## 跨标的总结

- 这轮没有超过 `round22`。统一口径下，全主题 `good/bad/unclear = 28/39/9`，方向准确率 `41.8%`。
- 分标的结果：
- `MU`: `8/15/3`，准确率 `34.8%`
- `AAOI`: `11/14/1`，准确率 `44.0%`
- `2513.HK`: `9/10/5`，准确率 `47.4%`
- 真实变化很清楚：`AAOI / 2513.HK` 基本保住甚至小幅改善，但 `MU` 被明显外溢伤害，所以这条“全局放宽催化后高波动整理型”的路不成立。

## 当前结论

- `round23` 证明了一个重要边界：即使结构化字段已经降级成辅助层，只要 tie-breaker 仍是全局生效，就依然会跨样本外溢。
- 因此后续结构化路线如果继续推进，修正必须变成更细的“ticker 语境内局部规则”，不能再使用跨样本共享的分型放宽。
