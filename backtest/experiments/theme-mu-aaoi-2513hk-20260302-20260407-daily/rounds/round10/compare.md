# round10

## 本轮目标

- 从 `round08` baseline 重新出发，只做一条很窄的修正：
- 针对高位强趋势、但短期高波动整理的样本，若报告核心建议仍是“不要追高，但回踩支撑或突破阻力后仍可顺势参与”，不要机械压成 `保持观望`。
- 这条修正的主要服务对象是 `AAOI`，但要求不能明显伤害 `MU` 与 `2513.HK`。

## 跨标的总结

- round10 在全主题准确率上超过了 round08。按统一的未来 3 日均价口径，全主题达到 `good/bad = 30/36`，方向准确率 `45.5%`，高于 round08 的 `41.3%`。
- `MU` 明显改善：从 round08 的 `9/11` 提升到 `10/10`，动作分布从 `10/10/6` 变成 `17/2/7`，执行均值也从 `-6.5776` 改善到 `-5.5415`。
- `2513.HK` 是本轮最大增量来源：从 round08 的 `9/11` 提升到 `14/8`，同时恢复到 `1` 笔交易、平均收益 `14.0`，说明这条窄规则意外地大幅改善了港股样本的方向质量。
- `AAOI` 没有按本轮原始意图改善，反而从 round08 的 `8/15` 退到 `6/18`。它仍然被 `hold` 和部分错误切换拖累，是这轮最主要的回撤来源。
- 结论：如果只看主题主指标“方向准确率”，round10 已经是新的最佳 round；但如果强调分标的均衡性，它仍带着明显的 `AAOI` 回撤，需要谨慎对待。
- 下一轮不应继续放大全局性改动。更合理的 round11 方向是：保留 round10 对 `MU / 2513.HK` 的改进，只针对 `AAOI` 做极窄修正，避免再次重伤这只样本。
