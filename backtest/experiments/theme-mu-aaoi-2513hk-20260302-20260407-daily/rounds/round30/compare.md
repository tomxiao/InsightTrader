# round30

## 本轮目标

- 保持 prompt 与 evidence schema 继续完全不变，只通过 `decision_rules.py` 验证一条更窄的残余修正规则。
- 本轮唯一实验假设是：
- `中性 + 暂不参与 + 催化后高波动整理型 + 边际走弱 + 高风险` 中，有一小批样本本质上仍属于高 beta 反弹前的等待型偏多，不应继续停留在 `hold`。

## 跨标的总结

- 这轮成功把主题主目标正式推过。统一口径下，全主题 `good / bad / unclear = 38 / 28 / 10`，方向准确率 **57.6%**。
- 这意味着：
- 明显高于 `round29` 的 **53.8%**
- 明显高于 `round13` 的 **49.3%**
- 首次正式超过主题目标 **55%**
- 分标的结果为：
- `MU`: `11/10/5`，**52.4%**
- `AAOI`: `13/11/2`，**54.2%**
- `2513.HK`: `14/7/3`，**66.7%**
- 这轮的提升几乎完全来自 `2513.HK`，但关键在于：`MU / AAOI` 都没有因此退步，所以这是一次非常干净的净改善。

## 结构化归因观察

- 三只票继续全部 `decision_source = rule_based`，说明到 `round30` 为止，这条路线的突破完全来自规则层，不再依赖 prompt 微调。
- `催化后高波动整理型` 的准确率从 `round29` 的 **46.4%** 提升到 **55.2%**，第一次跨过 `55%`。
- `中性` 的准确率进一步抬升到 **72.2%**，说明 round26 初期最保守、最拖累的中性分支，到这里已经被系统性修回。
- `暂不参与` 的准确率也从 `round29` 的 **40.0%** 提升到 **52.4%**，证明本轮命中的正是上一轮留下来的真正残余问题。

## 当前结论

- `round30` 成为当前主题的新总 baseline，也是第一轮正式达成 `55%` 目标的实验。
- 这条“证据层 + 规则决策层”路线已经完成从：
- `round26` 跑通链路
- `round27/28/29` 持续修回规则保守偏差
- 到 `round30` 正式突破主题目标
- 下一步不再适合继续盲目开新 round，更合理的是先把 `round30` 固化为新 baseline，再评估是否进入下一阶段：价格区间与执行质量优化。
