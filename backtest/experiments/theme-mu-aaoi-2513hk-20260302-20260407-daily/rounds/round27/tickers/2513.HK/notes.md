## 本轮目标

- 在不改 evidence schema 的前提下，修复 `2513.HK` 被机械压成 `中性 / hold` 的保守偏差。

## 改动内容

- 规则层新增了对 `中性 + 等待回调` 的有限放宽。
- prompt 与 evidence 抽取完全沿用 `round26`。

## 观察结论

- `2513.HK` 从 `round26` 的 `8/13/3` 提升到 `9/11/4`，准确率从 **38.1%** 提升到 **45.0%**。
- `Beta情绪技术驱动型` 的准确率在全主题从 **45.5%** 提升到了 **63.6%**，是本轮最亮眼的改善之一。
- 这说明对高 beta 样本来说，`中性 + 等待回调` 并不总是应该落到 `hold`；在当前 evidence schema 下，有限放宽这类组合是有效的。

## 下一步

- `2513.HK` 仍有不少 `中性 + 暂不参与` 的 bad hold，没有在本轮被触及。
- 如果继续推进规则层，下一步更值得研究的是：哪些 `暂不参与` 其实只是“高波动中的等待型偏多”，而不是应当继续保持 `hold`。
