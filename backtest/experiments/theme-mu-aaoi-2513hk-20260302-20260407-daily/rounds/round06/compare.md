# round06

## 本轮目标

- 继续只调 `decision manager`，不碰 analyst、parser 与 execution rules。
- 将 round05 的“技术计划不应被公司新闻静默机械否决”例外进一步收窄到极窄场景：
- 只有当当前价格已落入、逼近或正在测试首选参与区，且文本明确是“当前附近等待企稳参与”时，才允许从 `保持观望` 升到 `择机买入`。
- round06 的唯一目标是：在不伤 `MU / AAOI` 的前提下，尽量保住 `2513.HK` 的 signal quality 与 guardrail 作用。

## 跨标的总结

- 从动作分布看，round06 确实比 round05 更保守：`MU` 的 `buy` 从 `9` 降到 `6`，`AAOI` 从 `9` 降到 `5`，`2513.HK` 维持 `2`。
- 但从纯信号标签看，round06 没有带来正向改善：
- `MU`：`good/bad = 2/18`，比 round04 的 `7/13` 和 round05 的 `6/13` 都更差。
- `AAOI`：`good/bad = 3/21`，比 round04 的 `5/20` 更差，也没有优于 round05 的 `4/21`。
- `2513.HK`：`good/bad = 3/17`，比 round04/05 的 `4/16` 更差，且实际成交重新掉回 `0`。
- 结论：round06 证明“只靠进一步压窄港股例外规则”并不能得到我们想要的结果。它虽然收住了美股部分买点，但整体上把三只样本都往更差的方向推了，尤其重新引入了 `2513.HK` 的过度保守问题。
- 下一轮不应继续在同一条 prompt 边界上做更细碎的收缩。更合理的 round07 方向应是把优化目标拆成两层：先稳定纯信号方向，再单独处理价格区间与执行计划，不再让二者在同一轮里互相干扰。
