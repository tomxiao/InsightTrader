## 本轮目标

- 验证证据层与规则层能否更稳定地区分 `AAOI` 的“趋势延续”与“催化后高波动整理”，避免继续把它的后催化样本拍成单一 prompt 边界。

## 改动内容

- 改为由 `decision_rules.py` 基于证据卡决定最终动作。
- 重点观察 `方向偏向 / 风险状态 / 参与姿态` 是否能替代原先 prompt 里对 `AAOI` 的隐式判断。

## 观察结论

- `AAOI` 本轮是 `9/15/2`，准确率 **37.5%**，明显低于 `round24` 的 **44.0%**。
- 规则层对 `AAOI` 最大的问题不是过空，而是卖出严重不足：本轮只有 `1` 个 `sell`，同时产出了 `17` 个买入、`8` 个 hold。
- 证据卡显示 `AAOI` 被判成 `看多` 的有 `20` 次，`趋势完整性=完整` 的有 `23` 次，`参与姿态=等待回调` 的有 `21` 次。这说明第一版证据层把大量“催化后高波动整理”仍视为可继续回调参与，没能充分识别真正该转防守的样本。
- `催化后高波动整理型` 在全主题只有 **24.1%** 准确率，而 `AAOI` 正是这类问题最集中的样本。

## 下一步

- 后续如果继续沿证据层路线推进，`AAOI` 应该是第一优先级修正对象。
- 最值得修的不是全局 `sell` 规则，而是：当样本已处于“催化后高波动整理型”，且风险明显抬升时，什么时候应从 `等待回调` 切换到 `等待确认` 或 `暂不参与`。
