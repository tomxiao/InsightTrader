## 本轮目标

- 验证新的 evidence schema 是否能在 `2513.HK` 这类 beta / 技术驱动样本上避免再次因为“缺少公司催化”而过度保守。

## 改动内容

- 保留 `样本分型 / 主导驱动`，同时新增 6 个证据字段，并由规则层统一决定最终动作。
- 重点观察 `方向偏向`、`风险状态` 和 `参与姿态` 是否会把港股 beta 样本重新压回 `hold/sell`。

## 观察结论

- `2513.HK` 本轮为 `8/13/3`，准确率 **38.1%**，明显低于 `round24` 的 **50.0%**。
- 当前 evidence schema 对这类样本明显过于保守：`方向偏向=中性` 有 `12` 次，`风险状态=高` 有 `18` 次，`参与姿态=暂不参与` 有 `15` 次。
- 最终动作也因此被压成了 `13` 个 hold、`7` 个 sell、只有 `4` 个 buy，导致本轮虽然交易收益仍为正，但信号准确率明显退步。
- 这说明第一版规则层一旦读到“缺位催化 + 风险偏高 + 边际走弱”，就很容易把 `2513.HK` 压回 `中性/观望`，而没有给 beta 反弹样本保留足够的顺势参与空间。

## 下一步

- 若继续推进 round27，更值得优先修的是 `2513.HK` 与 `AAOI` 共通的保守偏置，而不是再动 `MU`。
- 具体方向应是：不要让 `催化缺位` 或 `风险偏高` 单独就把高 beta 样本机械推到 `中性 -> hold`。
