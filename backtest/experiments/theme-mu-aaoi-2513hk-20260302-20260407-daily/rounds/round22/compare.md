# round22

## 本轮目标

- 保留 `round21` 新增的结构化字段链路，但不再让它们充当全局总开关。
- 回到更接近 `round13` 的 signal-first 决策骨架，把 `样本分型 / 主导驱动 / 趋势完整性 / 风险状态` 降级成归因层与局部 tie-breaker。
- 验证结构化字段是否可以在不显著扩大 `hold` 滥用的前提下继续保留。

## 跨标的总结

- 这轮明显优于 `round21`，但仍低于 `round13`。统一口径下，全主题 `good/bad/unclear = 29/39/8`，方向准确率 `42.6%`。
- 分标的结果：
- `MU`: `9/14/3`，准确率 `39.1%`
- `AAOI`: `11/14/1`，准确率 `44.0%`
- `2513.HK`: `9/11/4`，准确率 `45.0%`
- 最重要的正向变化是：`趋势判断=震荡等待确认` 的数量从 `40` 次降到 `22` 次，说明结构化层造成的“观望泛滥”被明显压回去了。
- 但离 `round13` 的 `49.3%` 仍有差距，主要差距来自 `MU` 仍偏弱，以及 `催化后高波动整理型` 这类分型仍然容易落到低质量的 `hold/sell`。

## 当前结论

- 结构化字段值得保留，但只能作为辅助层。
- 当前这轮最有价值的经验是：一旦把结构化字段从“先分型再决策”改回“先信号判断再补结构化解释”，主指标会立刻回升。
- 下一步如果继续结构化路线，应该只针对最差分型 `催化后高波动整理型` 做更窄修正，而不是再碰全局框架。
