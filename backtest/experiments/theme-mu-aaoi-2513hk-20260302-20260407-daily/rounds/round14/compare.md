# round14

## 本轮目标

- 从 `round13` 出发，只修 `MU` 的健康回调误判。
- 核心假设：当报告已明确写成“财报前/宏观噪音下等待更好位置参与”，不应仅因不确定性就从 `趋势延续 -> 择机买入` 降到 `保持观望`。

## 跨标的总结

- `round14` 全主题达到 `good/bad = 31/34`，方向准确率 `47.7%`，低于 `round13` 的 `49.3%`。
- `MU` 明显修复，从 `8/12` 提升到 `10/11`；`AAOI` 也小幅改善到 `11/13`。
- 代价是 `2513.HK` 从 `15/8` 明显回落到 `10/10`，主要坏样本来自 `2026-03-04`、`2026-03-06`、`2026-03-30` 等原本正确的偏多样本被重新压成 `hold`。
- 结论：这条“财报前/事件前健康回调”guardrail 对美股强趋势样本有效，但外溢到港股高波动样本后副作用太大，不能直接作为全局规则放大。
- 下一轮应回到 `round13` 基线，把这条 guardrail 收窄成“只有上游报告形成更强共识时才生效”，验证能否保留 `MU` 改善同时修回 `2513.HK`。
