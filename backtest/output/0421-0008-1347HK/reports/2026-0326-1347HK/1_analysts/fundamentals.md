# 1347.HK 基本面分析报告

## 公司概况
华虹半导体是全球领先的特色工艺纯晶圆代工企业，专注于嵌入式/独立式非易失性存储器、功率器件、模拟与电源管理和逻辑与射频等特色工艺。

公司采用“8英寸+12英寸”、先进“特色IC+Power Discrete”的发展战略，服务于新能源汽车、物联网等新兴领域。

其在中国上海拥有三座8英寸晶圆厂，在无锡拥有两座12英寸晶圆厂，其中一条是全球首条12英寸功率器件代工生产线。

## 财务分析

### 资产负债表分析
截至2025年12月31日，公司总资产达10,159.3亿元，资产结构呈现重资产、高现金储备的特征。

**关键资产项目：**

1.  物业、厂房及设备：4,692.7亿元
    这意味着公司是典型的重资产制造业，庞大的固定资产是其核心产能的基础，但也带来了高额的折旧负担。

2.  现金及等价物：3,439.8亿元
    这意味着公司拥有极其充裕的现金储备，占总资产比例高达33.9%，为应对行业周期、持续资本开支及潜在风险提供了强大的安全垫。

3.  存货：382.6亿元
    存货规模相对可控，占流动资产比例约为8.2%，需关注未来需求变化可能带来的减值风险。

4.  应收账款：198.2亿元
    应收账款规模显著低于现金储备，回款风险相对较低，营运资金压力小。

**关键负债项目：**

1.  其他应付款及应计费用：708.9亿元
    这是最大的流动负债项，通常包含应付设备款、工程款及各项运营费用，反映了公司在建工程和日常运营产生的应付款项。

2.  短期贷款：283.8亿元
    相对于超过3,400亿元的现金储备，短期债务压力极小，偿债能力极强。

3.  应付账款：232.2亿元
    规模适中，表明公司在供应链中具备一定的账期优势。

**权益与偿债结论：**
公司权益基础雄厚，净流动资产高达3,339.0亿元，财务结构非常稳健。极高的现金/短期债务比率表明公司几乎无短期偿债压力，财务风险极低。

### 现金流量表分析
2025财年，公司现金流状况的核心特征是“经营造血能力强，但资本开支巨大”。

**经营现金流：**
经营业务现金净额为456.9亿元。
这意味着尽管利润表承压，但公司主营业务产生了强劲的现金流入，主要得益于巨额的非现金折旧摊销（521.4亿元），自我造血能力依然坚实。

**投资现金流：**
购建固定资产支出高达1,274.9亿元，是现金流出的绝对主力。
这意味着公司正处于积极的产能扩张周期，尤其是12英寸生产线的建设，投资活动现金流大幅净流出是战略投入期的必然结果。

**融资现金流：**
数据未完整提供，但从资产负债表看，公司主要依赖自有现金和经营现金流支持扩张，对外部融资依赖度低。

**现金净变动：**
在巨额资本开支下，公司期末现金仍较期初减少（数据未直接给出，但可从投资流出推断）。
这意味着公司正在消耗其庞大的现金储备以支持资本支出，现金余额从“极度充裕”向“充裕”水平回归，但仍远高于债务水平。

### 利润表分析
2025财年，公司面临收入下滑、毛利率收缩和净利润亏损的压力。

**收入表现：**
营业额为1,688.4亿元。
**注：** 由于缺乏去年同期对比数据，无法计算同比变化。但结合行业背景，该收入规模可能反映了半导体行业周期下行的影响。

**盈利能力：**
毛利为198.9亿元，毛利率约为11.8%。
毛利率处于较低水平，表明在行业需求疲软和产能爬坡阶段，公司面临定价压力和较高的固定成本摊薄压力。

经营溢利为-17.3亿元，除税前溢利为-62.2亿元。
这意味着公司主营业务已出现亏损，主要受高额折旧摊销及运营费用影响。

**费用结构：**
1.  行政开支：291.5亿元
    这是最大的费用项，远超销售费用，可能包含了研发投入、管理人员薪酬及与产能扩张相关的运营费用。
2.  销售及分销费用：7.7亿元
    占比较小，符合代工行业特性。
3.  融资成本：47.9亿元
    尽管现金充裕，但债务仍产生了显著的利息支出。

**亏损状况：**
税后溢利为-77.9亿元，但股东应占溢利为38.6亿元。
**关键点：** 净利润与归属股东净利润出现巨大背离，主要原因是“少数股东损益”为-116.5亿元。这意味着亏损主要由于并表子公司层面承担，而上市公司股东层面仍录得盈利。

每股基本盈利为0.225元。

## 关键财务比率分析
基于现有数据计算的关键比率如下：

1.  流动比率：流动资产/流动负债 = 46,437.2 / 13,047.0 ≈ 3.56倍
    短期偿债能力非常强劲。

2.  现金比率：现金及等价物/流动负债 = 34,397.6 / 13,047.0 ≈ 2.64倍
    仅现金就已能覆盖流动负债的2.6倍，流动性极度充裕。

3.  毛利率：毛利/营业额 = 198.9 / 1,688.4 ≈ 11.8%
    盈利能力当前承压明显。

4.  净利率（股东应占）：股东应占溢利/营业额 = 38.6 / 1,688.4 ≈ 2.3%
    扣除少数股东影响后，上市公司层面仍保持微利。

5.  固定资产占比：物业厂房设备/总资产 ≈ 46.2%
    重资产特征显著，经营杠杆高。

## 业务发展评估

### 积极因素
1.  **财务安全垫极厚**：超过3,400亿元的现金储备及极低的负债率，使公司能够安全度过行业寒冬，并逆周期进行战略投资。
2.  **特色工艺定位清晰**：在功率器件、嵌入式存储等细分领域具有领先地位，契合新能源汽车、绿色能源等长期增长赛道，差异化竞争避免与龙头在先进制程上直接血拼。
3.  **产能扩张有序推进**：12英寸产线的建设是提升长期竞争力、摊薄成本的关键。当前巨额的资本开支是为未来增长蓄力。
4.  **经营现金流健康**：在亏损情况下仍能产生超过450亿元的经营净现金流，证明其业务模式的现金生成本质未变，折旧摊销过后利润有弹性恢复空间。

### 风险因素
1.  **行业周期下行压力**：全球半导体行业需求疲软，可能导致产能利用率不足，持续压制毛利率和盈利能力。
2.  **巨额折旧摊销压力**：随着新产能投产，未来折旧摊销费用将进一步增加，若收入不能同步快速增长，将持续侵蚀报表利润。
3.  **资本开支消耗现金**：持续的巨额资本开支正在快速消耗现金储备，若行业复苏慢于预期，可能影响财务结构的长期稳健性。
4.  **子公司业绩拖累**：合并报表中少数股东承担巨额亏损，反映出部分子公司或新投产生产线面临严峻的经营挑战。

## 投资建议与风险提示

### 投资亮点
- **逆周期布局能力**：凭借无与伦比的资产负债表，公司有能力在行业低谷期持续投资，抢占市场份额，为下一轮周期复苏做好准备。
- **估值可能已反映悲观预期**：当前股价可能已充分计价了行业下行和短期亏损的利空，而对其特色工艺的长期价值和财务安全性定价不足。
- **短期催化剂关注现金流**：市场可能低估其强劲的经营现金流创造能力，这在熊市中尤为珍贵。

### 主要风险
- 行业复苏不及预期，导致产能利用率长期低迷。
- 新产线爬坡进度慢，折旧成本压力持续。
- 全球半导体供应链或地缘政治出现新的扰动。

### 交易建议
- **适合的投资者类型**：长线价值投资者、能够承受行业周期波动并相信中国半导体制造长期逻辑的投资者、寻找具备极高安全边际标的的防御型投资者。
- **不适合的投资者类型**：追求短期业绩高增长或趋势交易的投资者、对行业周期波动耐受度低的投资者。
- **持仓/建仓建议**：当前价位可考虑作为长期配置仓位分批建仓，但需做好承受短期股价波动和业绩压力的准备。不建议重仓博弈短期反弹。

## 结论
华虹半导体（1347.HK）是一家财务结构异常稳健、正处于战略投入期的特色工艺晶圆代工龙头。短期来看，行业下行周期与巨额资本开支导致其报表盈利能力承压；但长期而言，其深厚的现金储备、清晰的特色工艺定位以及逆周期扩张的能力，构成了显著的投资安全边际和长期增长潜力。当前股价更多反映了短期盈利困境，而对其资产负债表质量和长期竞争力定价不足。

## 机会表达
- **当前基本面是否支持未来1-3个月的估值修复或预期改善**：**有条件支持**。极度稳健的资产负债表和健康的经营现金流，使其在行业悲观情绪中具备防御性和估值修复的“安全”逻辑，但需等待行业触底或公司产能利用率改善的信号。
- **最强多头逻辑是什么**：**“破产风险为零”的极端安全边际叠加行业周期底部的逆周期投资能力**。市场恐慌时，其巨额现金和低负债成为稀缺资产，一旦行业预期边际改善，弹性可观。
- **最关键证伪点是什么**：**行业需求复苏持续低于预期，导致其12英寸新产线产能利用率长期无法有效提升**，从而使得巨额资本开支无法转化为盈利，持续消耗现金并拖累报表。
- **更像长期配置标的，还是阶段性交易机会**：**更像长期配置标的**。其投资逻辑的核心在于穿越周期，分享中国特色工艺半导体制造的成长，而非博弈短期的行业波动。
